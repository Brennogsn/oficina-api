# Oficina API

Sistema backend para gerenciamento de clientes, carros e serviços de uma oficina mecânica.

## Funcionalidades

- Cadastro de serviços
- Consulta de serviços por cliente
- Consulta de serviços por carro
- Remoção de serviços

## Tecnologias

- Node.js
- Express
- PostgreSQL

## Instalação

1 - Clonar repositório

git clone repositorio

2 - Instalar dependências

npm install

3 - Rodar servidor

node src/server.js
